# Quick Start

1. Start the API:
   ```bash
   pnpm dev:backend
   ```

2. Open `backend/bruno` in Bruno and select the local environment.

3. Put a real invite token in `invite_token`.

4. Run **Auth / Signup Whale User**.

5. Copy the response `token` to `auth_token`.

6. Use the market/admin endpoints. IDs returned by list endpoints should be
   copied into variables like `market_day_id`, `bet_id`, and `invite_id`.
